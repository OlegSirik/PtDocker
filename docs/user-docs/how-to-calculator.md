# Калькулятор страховой премии

## Назначение

Калькулятор страховой премии предназначен для настройки алгоритмов расчета страховой премии без разработки программного кода.

### Калькулятор позволяет:

определять страховые суммы и лимиты;
применять скидки и надбавки;
использовать тарифные таблицы;
использовать данные договора в расчетах.
Настройка выполняется через пользовательский интерфейс платформы либо с использованием встроенного AI-помощника.

### Как работает калькулятор
Во время расчета договора платформа последовательно выполняет формулы калькулятора.

Формулы могут использовать:

данные договора;
тарифные таблицы;
константы;
результаты предыдущих вычислений.

Результатом работы калькулятора является набор расчетных переменных, включая итоговую страховую премию.

Схема выполнения:

```
Данные договора
       |
       v
Тарифные таблицы
       |
       v
Формулы
       |
       v
Результаты расчета

```

### Типы элементов калькулятора
#### Тарифные факторы
Входные параметры представляют собой данные договора, используемые при расчете.

```
Примеры:

возраст застрахованного;
страна поездки;
страховая сумма;
срок страхования;
тип транспортного средства.
```

#### Константы (Constants)
Константы представляют собой фиксированные значения.
```
Примеры:

Код	Значение
BASE_RATE	0.005
MIN_PREMIUM	100
MAX_PREMIUM	50000
```
Константы используются для хранения редко изменяемых параметров продукта.

#### Тарифные таблицы (Rate Tables)
Тарифные таблицы позволяют определять значение по одному или нескольким параметрам договора.
```
Примеры использования:

возрастной коэффициент;
территориальный коэффициент;
базовый тариф;
скидка;
максимальная страховая сумма.
```
Пример таблицы:

Возраст от	Возраст до	Коэффициент
18	24	1.5
25	60	1.0
61	99	1.3
При расчете платформа автоматически определяет подходящую строку и использует её значение.

#### Переменные (Variables)
Переменные используются для хранения промежуточных результатов расчета.
```
Примеры:

базовая премия;
премия по покрытию;
скидка;
итоговая премия.
```
Переменные могут использоваться в других формулах.


#### Формулы (Formulas)
Формула определяет способ вычисления переменной.
```
Пример:

basePremium =
    sumInsured * BASE_RATE
или

premium =
    basePremium * ageFactor
```

Формулы выполняются последовательно сверху вниз.

#### Расчет по покрытиям
Для каждого покрытия используются отдельные переменные. Они создаются автоматически для каждого добавленного в пакет покрытия.

```
Например:

co_DAMAGE_premium
co_THEFT_premium
co_ACCIDENT_premium
```

После расчета премий по покрытиям вычисляется итоговая премия договора.
```
Пример:

policy_premium =
    co_DAMAGE_premium +
    co_THEFT_premium
```
#### Использование искусственного интеллекта

Калькулятор может быть настроен на естественном языке.
```
Пример запроса:

Для клиентов младше 25 лет увеличить премию на 10%. Для клиентов старше 65 лет увеличить премию на 20%.
```
После обработки запроса система автоматически создаст необходимые тарифные таблицы и формулы.
