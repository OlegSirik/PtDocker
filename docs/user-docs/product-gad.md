# Страхование бытовой электроники

## Общая информация

**Тип продукта:** страхование имущества
**Линия бизнеса:** страхование бытовой электроники

Продукт предназначен для страхования электронных устройств от поломок, случайных повреждений и других рисков, приводящих к утрате или повреждению устройства.

### Страхователь

физическое лицо
юридическое лицо

### Объект страхования

Объект страхования — имущественные интересы, связанные с владением и использованием электронного устройства, а также риском его:
* повреждения
* поломки
* утраты

## Страховая сумма

Страховая сумма определяется на основе стоимости устройства.

Возможные варианты:
### Фиксированная
```
Страховая сумма = стоимость устройства
```
### Диапазон страховой суммы
```
1 000 – 300 000 руб
```
(или в зависимости от категории устройства)

## Срок страхования

Типичные варианты:
* 1 месяц
* 1 год
* 2,3,5 лет
Срок страхования может совпадать со **сроком гарантии устройства**.

## Тарифная модель

### Фиксированная премия
Используется для массовых продуктов.
```
Страховая сумма: 1 000 000 руб
Премия: 200 руб
```

### Фиксированный тариф
```
Страховая сумма: 100 000 – 10 000 000
Тариф: 3%
```
Расчёт:
```
Premium = SumInsured * 0.03
```
### Тарифные таблицы (rating factors)

Тариф может зависеть от параметров устройства и условий страхования.

Пример:
```
К = К_срок * К_тип * К_стоимость
Premium = BaseRate * K
```

где:
* K_term — коэффициент срока страхования
* K_device_type — коэффициент типа устройства
* K_price — коэффициент стоимости устройства

## Тарифные факторы

Типовые тарифные факторы:
* тип устройства
* стоимость устройства
* срок страхования
* бренд устройства
* возраст устройства
