# Страхование грузоперевозок

## Общая информация

**Тип продукта:** страхование грузов

**Линия бизнеса:** cargo insurance

Продукт предназначен для страхования грузов на период перевозки от рисков утраты, повреждения или гибели в процессе транспортировки, перегрузки, хранения и иных связанных операций.

Страхование может распространяться как на разовую перевозку, так и на генеральный договор с декларированием отдельных отправок.

### Страхователь

* грузовладелец
* поставщик
* покупатель
* экспедитор
* перевозчик
* юридическое лицо
* физическое лицо / ИП

### Объект страхования

Объект страхования — имущественные интересы, связанные с владением, распоряжением и перевозкой груза.

Страхование распространяется на груз в процессе:

* транспортировки
* перегрузки
* временного хранения
* погрузо-разгрузочных работ

## Описание перевозки

Для расчета премии и оформления договора используются параметры перевозки.

### Описание груза

* тип груза
* описание груза
* состояние груза
* упаковка
* количество мест
* вес
* стоимость груза

### Описание транспортировки

* маршрут перевозки
* пункт отправления
* пункт назначения
* промежуточные пункты
* вид транспорта
* мультимодальная перевозка
* перевозчик
* экспедитор
* номер перевозочного документа

### Типы транспорта

Типовые варианты:

* автомобильный
* железнодорожный
* морской
* авиационный
* смешанный (multimodal)

## Страховые риски и покрытия

| Код покрытия | Покрытие | Список рисков |
|---|---|---|
| CARGO_ALL_RISKS | Страхование с ответственностью за все риски | DAMAGE, TOTAL_LOSS, PARTIAL_LOSS |
| CARGO_THEFT | Хищение груза | THEFT, ROBBERY, FRAUD |
| CARGO_LOADING | Риски погрузо-разгрузочных работ | LOADING_DAMAGE, UNLOADING_DAMAGE |
| CARGO_TEMPERATURE | Температурные риски | TEMPERATURE_CHANGE, REFRIGERATOR_FAILURE |

| CARGO_FIRE | Повреждение огнем | FIRE, EXPLOSION |
| CARGO_NATURAL_DISASTER | Стихийные бедствия | EARTHQUAKE, STORM, LIGHTNING |

| RATING_COVER | Техническое покрытие для расчета премии | (комплексное) |

## Страховая сумма

Страховая сумма определяется на основании стоимости груза.

Возможные варианты:

### Фиксированная

Страховая сумма = стоимость груза

### С учетом дополнительных расходов

Страховая сумма =
стоимость груза +
стоимость перевозки +
ожидаемая прибыль

## Лимит ответственности

* Лимит на одну перевозку
* Лимит на одно страховое событие
* Годовой агрегатный лимит

## Срок страхования

Типовые варианты:

* на период перевозки
* на конкретный рейс
* годовой договор
* генеральный полис

Период страхования может определяться:

* «от склада до склада»
* с момента погрузки до момента выгрузки

## Тарифная модель

### Фиксированный тариф

Страховая сумма: 10 000 000 руб  
Тариф: 0.15%

Расчёт:
```
Premium = SumInsured * 0.0015
```

### Тарифные таблицы (rating factors)

Тариф зависит от параметров груза и перевозки.

Пример:
```
K = K_transport * K_route * K_cargo * K_packaging
Premium = BaseRate * K
```

где:

* `K_transport` — коэффициент вида транспорта
* `K_route` — коэффициент маршрута
* `K_cargo` — коэффициент типа груза
* `K_packaging` — коэффициент упаковки

## Тарифные факторы

Типовые тарифные факторы:

* маршрут перевозки

* международная / внутренняя перевозка
* наличие перегрузок
* упаковка груза
* температурный режим
* опасный груз
* статистика убытков
* перевозчик
* срок перевозки

## Дополнительные условия

### Температурный режим

Для рефрижераторных грузов могут учитываться:

* температурные ограничения
* наличие рефрижераторной установки
* наличие температурного регистратора

### Опасные грузы

Для опасных грузов могут применяться:

* специальные тарифы
* ограничения покрытия
* повышенные франшизы

## Франшиза

Возможные варианты:

* безусловная франшиза
* условная франшиза
* процент от страховой суммы
* фиксированная сумма

## Документы

Типовые документы:

* страховой полис
* сертификат страхования
* генеральный полис
* счет на оплату
* кид
* приложение с перечнем перевозчиков
* приложение с маршрутами перевозки
