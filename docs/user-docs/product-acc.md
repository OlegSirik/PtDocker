## Тарифная модель
### Фиксированная премия
Страховая сумма: 1 000 000 руб
Премия: 200 руб
### Фиксированный тариф
Страховая сумма: 100 000 – 10 000 000
Тариф: 3%

Расчёт:
```
Premium = SumInsured * 0.03
```
### Тарифные таблицы (rating factors)

Тариф может зависеть от факторов риска.

Пример:
```
К = К_возраст * К_пол * К_спорт
Premium = BaseRate * K
```

## Тарифные факторы

Типовые факторы для НС:

* Возраст
* Пол
* Вид спорта
