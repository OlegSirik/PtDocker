# Страхование пассажиров

## Общая информация

**Тип продукта:** личное страхование
**Линия бизнеса:** страхование пассажиров

Продукт предназначен для страхования пассажиров на время перевозки и покрывает риски, связанные с несчастными случаями, травмами, медицинскими расходами, а также дополнительными рисками, связанными с перевозкой.

### Страхователь

физическое лицо

### Застрахованное лицо

физическое лицо

### Объект страхования

Объект страхования — имущественные интересы, связанные с:

* жизнью застрахованного лица
* здоровьем застрахованного лица
* расходами, возникшими вследствие несчастного случая во время перевозки

### Страховые риски и покрытия

| Код покрытия     | Покрытие                                             | Список рисков                        |
|------------------|------------------------------------------------------|--------------------------------------|
| PAX_ACCIDENT     | Несчастный случай с пассажиром                       | ACCIDENT_DEATH, ACCIDENT_DISABILITY  |
| PAX_DEATH        | Смерть пассажира в результате несчастного случая     | ACCIDENT_DEATH                       |
| PAX_DISABILITY   | Инвалидность пассажира вследствие несчастного случая | ACCIDENT_DISABILITY                  |
| PAX_INJURY       | Травмы пассажира                                     | ACCIDENT_INJURY                      |
| PAX_MEDICAL      | Медицинские расходы пассажира                        | MEDICAL_EXPENSE                      |
| RATING_COVER     | Техническое покрытие для расчета премии              |                                      |


## Страховая сумма

Возможные варианты:

### Фиксированная страховая сумма
Страховая сумма: 1 000 000 руб
### Диапазон страховой суммы
100 000 – 10 000 000 руб

## Срок страхования

Срок страхования соответствует периоду перевозки.

Полис действует:

* с момента посадки пассажира
* до момента завершения перевозки

## Тарифная модель

В продукте могут использоваться разные модели расчета страховой премии.

### Фиксированная премия
Страховая сумма: 1 000 000 руб
Премия: 200 руб

### Фиксированный тариф
Страховая сумма: 100 000 – 10 000 000
Тариф: 3%

Расчёт:
```
Premium = SumInsured * 0.03
```
### Расчет по количеству сегментов перевозки (legs)

Используется для маршрутов с несколькими участками перевозки.
```
Premium = legs * base
```
где:

* legs — количество сегментов маршрута
* base — базовая ставка

### Расчет от стоимости билетов

Используется при интеграции со системами продажи билетов.
```
Premium = TicketPrice * Coefficient
```
где:

* TicketPrice — стоимость билета
* Coefficient — тарифный коэффициент

## Тарифные факторы

В зависимости от конфигурации продукта тариф может зависеть от следующих факторов:

* стоимость билета
* количество сегментов маршрута
