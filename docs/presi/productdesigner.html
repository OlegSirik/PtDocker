Основа продуктовой линейки

Product Family объединяет страховые продукты одной линии бизнеса.

Travel Insurance Motor Insurance Property Insurance Accident Insurance

В рамках Product Family настраиваются общие компоненты:

Страхователь, застрахованные лица, объекты страхования и другие сущности.

🛡️ Покрытия и риски

Единый каталог покрытий для всей линейки продуктов.

📊 Тарифные таблицы

Переиспользуемые коэффициенты и тарифы.

🧮 Формулы

Общие элементы расчетов.

📄 Документы

Шаблоны страховых документов.

Такой подход позволяет создавать новые продукты значительно быстрее.

Настройка продукта

Каждый страховой продукт создается на основе Product Family.

Для продукта определяются:

  • код продукта
  • название
  • страховая компания
  • доступные покрытия
  • тарифные факторы
  • правила проверки
  • документы
  • калькулятор
Product Family │ ▼ Product │ ▼ Version
Каждый продукт может иметь несколько версий, что позволяет безопасно изменять конфигурацию без влияния на действующие договоры.
03 / 10

Конструктор схемы договора

Contract Schema Designer позволяет описывать структуру страхового договора.

Пользователь может:

  • удалять ненужные поля
  • добавлять новые атрибуты
  • настраивать обязательность
  • определять тарифные факторы
  • ограничивать допустимые значения

Примеры объектов

Policy Policy Holder Insured Person Vehicle Property Beneficiary
Схема договора используется во всех компонентах платформы: калькуляторе, проверках, андеррайтинге, документах, API.

Покрытия и риски

Покрытия являются основой страхового продукта.

Для каждого покрытия задаются:

  • код
  • наименование
  • описание
  • параметры
  • правила включения

Примеры покрытий:

Medical Expenses Trip Cancellation Vehicle Damage Theft Personal Accident

💰 Собственные тарифы

Каждое покрытие может иметь собственные тарифы, лимиты и правила расчета.

📦 Пакетные предложения

Покрытия могут объединяться в пакеты, упрощая настройку продуктовых предложений.

Пользователь может настраивать:

  • тарифные факторы
  • константы
  • тарифные таблицы
  • формулы
  • коэффициенты
  • скидки и надбавки

Правила и андеррайтинг

Платформа поддерживает настройку бизнес-правил на всех этапах жизненного цикла договора.

📋 Pre-Quote

Проверка входных данных.

📬 Post-Quote

Автоматическое заполнение данных.

⚙️ Pre-Calculate

Подготовка данных для расчета.

📊 Post-Calculate

Применение скидок и надбавок.

✅ Underwriting

Автоматическое принятие решения.

Правила настраиваются через визуальный интерфейс и могут использовать данные договора и результаты расчета.

Конструктор документов

Document Designer позволяет создавать и поддерживать страховые документы без разработки.

Поддерживаются:

  • страховой полис
  • заявление
  • приложения
  • сертификаты
  • KID
  • коммерческие предложения

Шаблоны используют:

  • данные договора
  • результаты расчетов
  • шаблонные строки
  • правила продукта
Изменение документа выполняется через интерфейс и не требует релиза системы.
