# PxP Policy.Tech Direct Insurance Platform

Low-code платформа предназначена для **быстрого запуска и масштабирования страховых и сервисных продуктов** через партнёрские каналы, агрегаторы и API-интеграции.

Платформа позволяет страховым компаниям создавать и изменять страховые продукты без разработки и выпуска новых версий программного обеспечения. Структура договора, тарифные калькуляторы, правила андеррайтинга, печатные формы, клиентские сценарии и бизнес-правила настраиваются через конфигурацию и метаданные.

Платформа интегрирована с технологиями искусственного интеллекта, что позволяет бизнес-пользователям создавать и изменять правила, калькуляторы, проверки, документы и продуктовые настройки на естественном языке. AI преобразует бизнес-описание в исполняемую конфигурацию платформы, значительно сокращая время вывода новых продуктов на рынок.

Платформа ориентирована на **direct-продажи, embedded insurance и партнерские каналы**.


## Основные возможности

* Каталог страховых продуктов и версионирование
* Low-code настройка продуктовой модели
* AI-помощник для создания правил и калькуляторов на естественном языке
* Гибкий движок расчета премии
* Настраиваемые правила андеррайтинга
* Динамическая структура договора страхования
* Генерация документов и печатных форм
* Настраиваемые пользовательские сценарии и вопросники
* Поддержка прямых каналов продаж
* Отчетность и операционные дашборды
* Мультитенантная архитектура
* API для интеграции с внешними системами
