# Доменная модель — обзор

## Назначение
Данный раздел описывает бизнес-домен системы страхования,
его ключевые понятия, правила и ограничения.

Доменная модель:
- независима от API
- не зависит от persistence
- является источником бизнес-правды

## Основные принципы
- Domain-Driven Design (DDD)
- Явные инварианты
- Богатая модель (rich domain model)
- Чёткое разделение domain / application / infrastructure
