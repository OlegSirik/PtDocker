# Словарь терминов

## Цель
Единый язык используется:
- в коде
- в документации
- в обсуждениях с бизнесом

## Основные термины

| Термин | Описание |
|------|----------|
| Policy | Договор страхования |
| Product | Страховой продукт |
| Coverage | Покрытие |
| Insured Object | Объект страхования |
| Asset | Материальный или нематериальный объект |
| Premium | Страховая премия |
| Sum Insured | Страховая сумма |
| Claim | Страховой случай |
| Endorsement | Изменение договора |

## Правила
- Термины не переводятся внутри домена
- Синонимы запрещены
- UI и API используют те же названия
