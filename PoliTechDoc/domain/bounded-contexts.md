# Bounded Contexts

## Контексты системы

| Контекст | Назначение |
|--------|-----------|
| Product | Конфигурация страховых продуктов |
| Policy | Жизненный цикл договоров |
| Pricing | Расчёт премии |
| Payment | Платежи и комиссии |
| Claims | Урегулирование убытков |
| Printing | Генерация печатных форм |
| Authorization | Права доступа |
| Tenant | Управление tenant |

## Взаимодействие
- Контексты взаимодействуют через API
- Прямой доступ к данным запрещён
- ACL используется при интеграции

## Ownership
Каждый контекст владеет своей моделью и правилами.
