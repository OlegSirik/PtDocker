# Коды разрешений

## Назначение
Этот документ определяет канонический список кодов разрешений, используемых во всей системе.


Коды разрешений являются:
- Стабильными
- Версионируемыми
- Определенными в коде (enum)

## Формат
<ресурс>:<действие>

##Примеры
### Полис
~~~
policy:create
policy:read
policy:update
policy:issue
policy:print
~~~

### Клиент
~~~
client:create
client:read
client:update
~~~

### Платеж
~~~
payment:create
payment:refund
~~~

## Управление
- Новые коды разрешений требуют ADR
- Устаревшие коды поддерживаются как минимум один релиз
